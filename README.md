## EFD

EFD stands Eddie's Frontend Development, which personally used.

## What is EFD?

EFD use pros of 4 development techniques such as ,CDD, TDD, SDD and BDD. 

## Start

```bash
cd EDD

# START SERVER
cd back
npm install
npm run start

# START FRONT
cd front
npm install

# FRONT PROJECT
npm run serve 

# STORYBOOK
npm run storybook

# UNIT TEST
npm run test:unit:watch

# CYPRESS
npm run test:e2e
```
